## Where is it from

[MOJ filter](https://design-patterns.service.justice.gov.uk/components/filter/)

The CPS filter is based on the MOJ filter component. Current implementations do not use the selected filters element at the top.

## How it works

The filter component can use form controls such as radio buttons, checkboxes, date inputs and text boxes.

Users can select one or more filters and submit the form. The page refreshes to show the items that match the filters.
